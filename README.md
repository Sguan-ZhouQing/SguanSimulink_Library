# SguanSimulink库
这是一个专为电机控制系统开发设计的综合仿真平台，基于磁场定向控制（FOC）原理构建，为永磁同步电机（PMSM）控制系统开发提供从模块级验证到系统级仿真的全流程数字仿真环境。

## 🎯 核心定位
为电机控制领域工程师提供从单个控制模块实现到大规模 PMSM 系统验证的完整数字仿真环境，专门针对实际工程场景定制，支持基于 DSP/STM32 控制器的自动代码生成，实现仿真到实际产品部署的无缝过渡。

## 🏗️ 架构设计
项目采用**双层模块化架构**，将基础控制理论模块与复杂系统级仿真模型解耦，开发者可先独立测试验证各组件，再集成到完整电机控制系统中，大幅降低开发风险与调试复杂度。

| 目录名称 | 用途 | 内容类型 | 目标用户 |
|----------|------|----------|----------|
| SguanSimulink模块库 | 基础控制模块开发 | 独立 .slx 模型 + C 源代码 | 算法开发者、控制工程师 |
| SguanSimulink模型库 | 系统级仿真验证 | 完整的 PMSM 控制系统 | 系统架构师、验证工程师 |

## ✨ 核心能力
- **部署前风险评估**：硬件实现前完成全面故障仿真（逆变器开关故障、编码器异常、电流采样偏差等）和稳定性边界测试（突加负载、速度跳变、参数变化），识别系统不稳定性阈值，避免后期设计返工。
- **全生命周期支持**：覆盖从组件设计、模块验证到系统部署的完整电机控制开发生命周期。

## 🚀 未来规划
我们将持续迭代更新，不断完善功能模块，致力于打造一个功能强大、贴合工程实际的 PMSM 永磁同步电机仿真模型库。
