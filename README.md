# SguanSimulink_Library
用于电机控制开发的 Simulink 项目仿真文件，基于自主开发的磁场定向控制（Field-Oriented Control, FOC）仿真库构建，为电机驱动系统的设计、验证与优化提供完整的数字化仿真平台。该仿真套件专为实际工程环境定制，可在产品开发早期阶段实现高精度预测与系统性验证。

核心功能与应用价值
1. 预先问题检测与风险评估
系统级故障模拟：支持逆变器故障（如开关管开路/短路）、传感器异常（编码器失效、电流采样偏差）、电源扰动等典型故障场景的注入与影响分析

稳定性边界测试：自动扫描负载突变、转速跳变、参数摄动等工况，识别系统失稳临界点

热性能与应力评估：集成损耗计算模型，预测功率器件温升与热循环应力，优化散热设计

2. 多层级 PID 参数调节与优化
分层调节架构：

电流环调节：支持d轴/q轴电流PI参数自动整定，兼顾动态响应与抗扰性能

速度环调节：提供基于模型参考自适应（MRAS）与频域分析的参数优化流程

位置环调节：针对伺服系统，整合前馈补偿与陷波滤波器协同设计

在线调参工具箱：集成Ziegler-Nichols、CHR、IMC等多种整定算法，支持实时参数迭代与性能指标（IAE、ITSE）可视化评估

3. 高保真仿真模型库
电机本体模型：

永磁同步电机（PMSM/IPMSM）非线性磁链模型，考虑饱和与交叉耦合效应

感应电机（IM）的多参数动态模型，支持转子电阻在线辨识

功率变换器模型：

开关器件非线性模型（导通压降、开关延时、死区效应）

母线电容老化模型与直流链路电磁兼容（EMC）特性模拟

实时控制硬件在环（HIL）接口：

支持TI C2000系列（如DSP28335/28379D）、STM32系列控制器代码自动生成与验证

提供FPGA-based高速仿真接口，适用于多核并行计算与超高频开关模拟
